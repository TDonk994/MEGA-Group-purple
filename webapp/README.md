# webapp-deface
 <b>web application built in python and docker-compose for the purpose of being defaced</b>

<b>seperate repository:</b>>
https://github.com/hankvatfleming/webapp-deface

 <b>do not edit the requirements.txt file</b>

 <b>your directory should consist of these items together to ensure functionality:</b>
 
 app.py

 docker-compose.yml

 Dockerfile

 requirements.txt

 <b>to launch the webapp run:</b>

 sudo docker-compose up -d

 <b>to kill the webapp:</b>

 sudo docker-compose down
